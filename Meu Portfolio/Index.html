Latest Projects

Projeto Mangas Soword

Meu primeiro projeto foi uma loja online chamada Mangas Soword, criada para explorar e aplicar os conceitos aprendidos no curso de desenvolvimento web. Nesse projeto, desenvolvi uma página estilizada com HTML e CSS, destacando os produtos de maneira atraente e funcional. Além disso, utilizei JavaScript para implementar interatividade, como um sistema de navegação dinâmico e elementos que melhoram a experiência do usuário. Esse projeto foi um marco na minha jornada, pois me permitiu colocar em prática as habilidades aprendidas e entender como estruturar e estilizar uma aplicação web..

Projeto Bolos Unidos

Meu segundo projeto foi uma loja online chamada Bolos Unidos, criada para expandir minhas habilidades em desenvolvimento web. Nesse projeto, me concentrei em criar uma interface amigável e responsiva, utilizando HTML, CSS e JavaScript. Implementei funcionalidades como a exibição de produtos organizados, menus interativos e um layout que se adapta a diferentes dispositivos, garantindo uma navegação agradável tanto em desktops quanto em smartphones. Esse projeto reforçou minha compreensão sobre a importância da usabilidade e do design, além de aprimorar minhas técnicas de organização e estruturação do código.

Projeto Ice Friends

Meu terceiro projeto foi uma sorveteria virtual chamada Ice Friends, onde busquei aprimorar ainda mais minhas habilidades no desenvolvimento web. Nesse projeto, criei uma interface colorida e convidativa, refletindo a leveza e diversão associadas ao tema. Utilizei HTML e CSS para estruturar e estilizar o site, além de JavaScript para adicionar interatividade. Esse projeto me ajudou a explorar novos elementos de design e a trabalhar com conceitos que focam na experiência do cliente, tornando-o mais imersivo e atrativo.
